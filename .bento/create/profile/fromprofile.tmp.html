This option is used to create extrusions based on a profile which is defined in a parametric manner. When you press this option, displays the parametric profile shapes you have already created:

The buttons on the right are explained below:

New

You can sketch a completely new profile by touching New. The profile starts with a single straight section, but you can then add new sections and modify the lengths of sections and the angles between sections.

Edit

Select one of these profiles by touching it, and then touch Edit to modify it for use. From this, editor window, you can then set up a suitable material and thickness and extrude the profile to create a workpiece.

Clone / Delete

Use Clone to make a copy of the selected profile (so you can then modify the copy), or touch Delete to delete the selected profile.

Rename

Click on the Rename button and it opens a dialog (see picture below) to rename the selected profile.

Extrude

Select an existing profile and then click Extrude. The part settings dialog pops up where the value for Extrusion can be entered and then press the Done button to create it. See the section on Part Settings for more details.

Import

Import a dxf file which contains the side view of the part.
