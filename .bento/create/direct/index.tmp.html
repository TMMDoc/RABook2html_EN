RightAngle offers the functionality to direct program all motions of the machine. The Direct Program environment is used to create a simple bending program directly on the control. The program is created by specifying the bend length, the bend angle and the gauges positions for each bend. If values outside expected ranges are entered, the input is refused, and the expected range is displayed (in the top right corner). The error message is transient - it disappears after a few seconds.

Important
 To understand the Direct Programming you need to have a clear understanding of the axes used: Gauging

The regular way to create a new extrusion is

  1. pres the 'new' button to design a new profile or edit a existing one

  2. extrude the profile

  3. add the technology information with the Setting dialog
