The main screen is composed of the following sections:

  • Settings: This section contains the settings for the current operation. You can change the settings by clicking on the Setting button.

  • Bends: To start programming you need to add your first bend

  • Bend step monitor: This section shows the current bend step and the total number of steps

  • Bend step: To navigate to the next step navigate

  • Jog: Press the Jog button to bring up the Axis Jog Panel. This allows the user to freely move all the gauge axes using either the handwheel, or by holding down the jog buttons. From this panel, one can record the position of the gauges directly into the program by tapping the teach button there.
